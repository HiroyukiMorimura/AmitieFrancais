# アミティエ学習アプリ 🇫🇷

フランス語学習を楽しく続けるための Web アプリケーション

## 概要

このソフトウェアは、フランス語学習者が実践的なスキルを身につけられる学習支援アプリケーションです。

## 主な機能

- 📰 **時事単語** - ニュースに登場する重要単語を学習
- 📝 **名詞化ジム** - より洗練されたフランス語表現を習得
- 🔤 **動詞ジム** - 動詞の活用と時制を徹底トレーニング
- ✍️ **仏作文** - 日本語からフランス語への翻訳練習
- 📚 **仏検対策** - 実用フランス語技能検定試験対策
- 📊 **学習記録** - 学習時間、正答率、連続学習日数を自動記録

## 使い方

### 1. 初めて利用する方

1. アプリケーションを開く(https://amitie-francais.vercel.app/)
2. 「アプリを始める」→「アカウントを新規作成」をクリック
3. メールアドレスとパスワードを入力して登録
4. 登録後、Supabase Auth<noreply@mail.app.supabase.io>からメールが届くので、メールを確認して、メール内のリンク"Confirm your mail"をクリックして、アカウントを有効化してください。  
5. https://amitie-francais.vercel.app/に戻り、「アプリを始める」を押して下さい。その後ログインします(2.ログインへ)

### 2. ログイン

1. メールアドレスとパスワードを入力
2. 「ログイン」をクリック

### 3. 学習モジュールを選ぶ

ダッシュボードから学びたいモジュールを選択してください：

**時事単語**
- カードをめくって単語と意味を確認
- 「覚えた」「まだ」ボタンで学習を記録

**名詞化ジム**
- 動詞や形容詞を名詞形に変換する練習
- 答えを確認して正解/不正解を記録

**動詞ジム**
- 指定された時制で動詞を活用
- 答えを表示して学習を進める

**仏作文**
- 日本語の文章をフランス語に翻訳
- 模範解答を確認

### 4. 学習記録を確認する

- **ダッシュボード**: 学習時間、連続日数、総正答数を表示
- **レポート画面**: 詳細な学習データとグラフを確認

### 5. ショートカットキー ⌨️

学習モジュール（時事単語、名詞化ジム、動詞ジム、仏作文）では、以下のキーボードショートカットが利用できます：

| キー | 機能 |
|------|------|
| **Space（スペース）** | 答えを表示 / 次のカードへ進む |
| **S** | 正解としてマーク（答え表示後のみ） |
| **F** | 不正解としてマーク（答え表示後のみ） |
| **C** または **→（右矢印）** | 次のカードへ進む |
| **X** または **←（左矢印）** | 前のカードに戻る |

> **ヒント**: ショートカットキーを使うことで、マウスを使わずにスムーズに学習を進められます！

## よくある質問

**Q: パスワードを忘れました**
A: ログイン画面から「パスワードをお忘れですか？」をクリックして、パスワードリセットメールを送信できます。

**Q: 学習データは保存されますか？**
A: はい、すべての学習記録は自動的に保存されます。

**Q: 複数デバイスで利用できますか？**
A: はい、同じアカウントでログインすれば、どのデバイスからでも学習データにアクセスできます。
